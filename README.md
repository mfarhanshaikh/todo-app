## 🌟 Key Features

### 💎 Premium Interface
- **Modern Glassmorphism**: High-end transparent layers with multi-level backdrop blur effects.
- **Dynamic Themes**: Seamless Dark/Light mode switching with automatic system detection.
- **Custom Priority Component**: A bespoke, high-precision selector for Low, Medium, and High priority tasks.
- **Cinematic Animations**: Zero-jitter transitions and elegant slide-in effects.

### ⚡ Productivity Optimized
- **Smart Logic**: Full Add, Edit, and Delete workflow with instant visual feedback.
- **Advanced Filtering**: Categorize tasks by *All*, *Active*, or *Completed* states.
- **Bulk Cleanup**: Single-click "Clear Completed" with safety confirmation dialogs.
- **Privacy Toggle**: Easily hide or show completed tasks to declutter your workspace.

### 📱 Edge-to-Edge Responsiveness
- **Adaptive UX**: Designed for every viewport—from the smallest mobile to large 4K displays.
- **Persistent State**: Full `LocalStorage` synchronization for tasks, filters, and theme preferences.
- **Layout Safety**: Built-in word-wrapping and overflow protection for long task entries.
